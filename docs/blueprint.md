# **App Name**: PDF Quote Generator

## Core Features:

- PDF Template Upload: Allow users to upload a PDF template with designated spaces for information.
- Input Fields: Provide input fields for 'Cliente,' 'Contacto,' 'Fecha de cotizaci√≥n,' item descriptions, and prices.
- Auto-Generate Quotation Number: Automatically generate a unique quotation number for each new quotation.
- Dynamic Item Input: Enable users to add multiple items (services) with descriptions and prices.
- Discount Option: Implement a checkbox to enable or disable discount and a field for discount percentage; discount information should not appear in the generated PDF if the checkbox is not selected.
- Automatic Calculation: Automatically calculate 'Total sin Descuento,' 'Total con Descuento,' 'IVA,' and 'TOTAL' based on the inputs.
- PDF Generation: Generate the final quotation in PDF format, filling in the information into the template provided.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db), representing professionalism and reliability.
- Background color: Light gray (#ecf0f1), offering a clean and neutral backdrop.
- Accent color: Emerald green (#2ecc71), used for success indicators or 'calculate' buttons.
- Body font: 'Inter', sans-serif, for clean and modern text.
- Headline font: 'Space Grotesk', sans-serif, to create prominent headlines with a scientific feel.
- Use minimalistic icons for various UI elements (e.g., a document icon for PDF download).
- Maintain a clean and structured layout to match the design of professional quotation documents.
- Subtle animations for actions like calculating totals or generating the PDF.
